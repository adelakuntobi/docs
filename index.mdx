## What PASCA is

PASCA is an eInvoicing platform that helps you create, sign, and transmit compliant invoices while keeping your business aligned with NRS requirements. You use PASCA to standardize invoice data, validate it before submission, and track every invoice through fiscalization.

## What the API does

The PASCA API gives you programmatic access to core eInvoicing workflows:

- Validate invoice payloads before submission
- Sign invoices to generate compliant IRNs
- Transmit invoices for fiscalization
- Retrieve invoice status and verification results
- Access reference data such as tax categories and units of measurement

Start with authentication, then follow the core modules to understand the end-to-end flow. Each endpoint page includes request and response examples plus field-level explanations so you can integrate quickly and safely.
