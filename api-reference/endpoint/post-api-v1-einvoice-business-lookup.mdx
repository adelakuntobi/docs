**POST** `/api/v1/einvoice/business/lookup`

## Request example

```bash
curl -X POST "{{BASE_URL}}/api/v1/einvoice/business/lookup" \
  -H "API-KEY: <your_api_key>" \
  -H "API-SECRET: <your_api_secret>" \
  -H "Accept: application/json"
```

Request body:

```json
{
  "tin": "32637045-0001"
}
```
