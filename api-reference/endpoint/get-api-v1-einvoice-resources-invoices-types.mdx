**GET** `/api/v1/einvoice/resources/invoices-types`

## Request example

```bash
curl -X GET "{{BASE_URL}}/api/v1/einvoice/resources/invoices-types" \
  -H "API-KEY: <your_api_key>" \
  -H "API-SECRET: <your_api_secret>" \
  -H "Accept: application/json"
```
