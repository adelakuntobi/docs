- `invoice_line`: Line items for products or services.
- `legal_monetary_total`: Invoice totals.

## Transmit invoice

**GET** `{{BASE_URL}}/api/v1/einvoice/transmit/{IRN}`

Transmits a previously signed invoice using its IRN.

Request example:

```bash
curl -X GET "{{BASE_URL}}/api/v1/einvoice/transmit/{{IRN}}" \
  -H "API-KEY: <your_api_key>" \
  -H "API-SECRET: <your_api_secret>" \
  -H "Accept: application/json"
```

Response format:

```json
{
  "status": true,
  "code": 200,
  "message": "Invoice transmitted successfully",
  "data": []
}
```

## Update invoice

**PATCH** `{{BASE_URL}}/api/v1/einvoice/update/{IRN}`

Updates the payment status or reference for a stored invoice.

Request example:

```json
{
  "payment_status": "REJECTED",
  "reference": "payment_reference_or_note"
}
```

Response format:

```json
{
  "status": true,
  "code": 200,
  "message": "Invoice updated successfully",
  "data": []
}
```

Fields:

- `payment_status`: One of `PENDING`, `PAID`, or `REJECTED`.
- `reference`: Optional payment reference or note.

## Report e-invoice

**POST** `{{BASE_URL}}/api/v1/einvoice/report-fiscalization`

Reports a fiscalized invoice and its tax details.

Request example:

```json
{
  "agent_tin": "29039239-0001",
  "base_amount": "1000",
  "beneficiary_tin": "29334238-0001",
  "currency": "NGN",
  "item_description": "Items",
  "irn": "{{IRN}}",
  "other_taxes": "5000",
  "total_amount": "112500",
  "transaction_date": "2024-11-18",
  "vat_calculated": "7500",
  "vat_rate": "7.5",
  "vat_status": "STANDARD_VAT"
}
```

Response format:

```json
{
  "status": true,
  "code": 200,
  "message": "Report submitted successfully",
  "data": []
}
```
